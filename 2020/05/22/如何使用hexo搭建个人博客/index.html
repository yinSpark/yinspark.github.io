一、搭建本地写作环境

1.1 环境配置

使用 Hexo 必需先安装 Node.js 和 Git。
安装 Hexo，命令
第一步：使用 cnpm，安装淘宝镜像源
sudo npm install -g cnpm --registry=https://registry.npm.taobao.org
第二步：安装hexo脚本架
sudo cnpm install hexo-cli -g

    tree -L 1
  3. 启动服务，本地预览
    hexo server 或简化的 hexo s
